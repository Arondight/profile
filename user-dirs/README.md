## user-dirs

Default XDG user directories, configured via `user-dirs.dirs` and applied by `xdg-user-dirs-update`.

| Name      | Path            |
| --------- | --------------- |
| Desktop   | `~/Desktop`     |
| Download  | `~/Download`    |
| Templates | `~/Templates`   |
| Share     | `~/Publicshare` |
| Documents | `~/Documents`   |
| Music     | `~/Music`       |
| Pictures  | `~/Pictures`    |
| Videos    | `~/Videos`      |
