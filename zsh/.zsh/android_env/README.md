你还需要设置以下软连接：

```
sudo ln -sf /usr/lib/libncursesw.so /usr/lib/libtinfo.so.6
sudo ln -sf /usr/lib/libncursesw.so /usr/lib/libtinfo.so.5
sudo ln -sf /usr/lib/libncursesw.so /usr/lib/libtinfo.so
sudo ln -sf /usr/lib/libncursesw.so /usr/lib/libncurses.so.5
sudo ln -sf /usr/lib/libncursesw.so /usr/lib/libncurses.so.6
```
